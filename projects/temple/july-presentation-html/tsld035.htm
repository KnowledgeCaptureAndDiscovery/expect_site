Linking the critiquer to a planner can be easy or hard.

  • “passenger capacity must be greater than 20” can be used as advice for a planner (as in PASSAT). It constrains valid selections for an object used in the plan.

  • “only drive if driving time would be less than 3 hours” can be used as advice for a planner. It constrains choice of action, so it is harder to show to a user.

  • “pick a hotel within walking distance, otherwise rent a car” could be used as advice, but should be applied to more than one decision point in the planner.
