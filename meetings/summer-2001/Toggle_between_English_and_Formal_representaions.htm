1. MethodList Panel

The MethodList Panel contains list of methods in formal representation.

Currently there are 3 views of the same method in English

  • Show Method In English – This brings up another window showing the English description.Shows the method as unformatted lines of text.
  • Modify – Brings up the Structured Editor.The method is divided into 3 parts-

Capability

Method

Result Type
  • Modify in NL - Brings up the Method Editor which brings up the method as In order to …Returning format.
  • The English description for each method appears to be quite verbose. It should be rendered before display. My suggestion is to have a have a collapsed tree format with the capabilites as the nodes. The user can click on the capability to get a detailed description.
  • The Method List panel contains a 'Method Organizer' to copy, paste, create a new folder etc. These functions are performed on the rendered formal method representations.This might require some amount of work to get all of them working with the English representations.
  • The 'Edit Methods' portion contains buttons to add new method , copy ,Modify which bring up the Structured Editor and the Modify in NL brings up the Method Editor.
