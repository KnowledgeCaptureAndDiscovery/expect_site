In entering specific models such as Irak’s, the user may first select the relevant process models that they beliwve Irak is considering (e.g., Anthrax, etc.).  See previous slide.
Then, the user would specify what evidence they have about Irak having specific resources to implement those process models.  For example, here the user is specifying how Irak can obtain two key ingredients for Anthrax production.  Notice that this kind of knowledge could be stated by the user  in some form of restricted English, and the system could use the models that the user has previously indicated as a guide to NLP.
