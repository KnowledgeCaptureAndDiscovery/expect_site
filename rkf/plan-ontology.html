Plan and Process Ontologies

  • PSL : the Process Description Language is a standard developed by NIST.
  • SPAR : the Planning Initiative's Shared Plan and Activity Representation.
  • PIF: ontology of the Process Interchange Format working group.
  • CPR: Common Plan Representation developed for DARPA's JTF/ATD.
  • EO: the Enterprise Ontology defines activities and processes within business organizations.
  • ACT: SRI's formalism to represent procedural knowledge for planning and execution.
  • PDDL : the Planning Domain Definition Language used in the AIPS planning competition.
  • PLANET : ISI's ontology to represent planning knowledge inspired by planning research. It is available in Loom, KIF, and MELD. Documentation about its alignment with IKB during the HPKB program is available at http://www.isi.edu/isd/HPKB/planet/alignment.
