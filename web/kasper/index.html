Knowledge Acquisition for Solving Problems

Description

We are participating in the DARPA Rapid Knowledge Formation (RKF) program. As a part of the program we are working on several research thrusts (1) KANAL: building/checking process models (2) Dialog Manager: keeping track of user actions and making suggestions (3) Expression Composer: helping users to express their knowledge without worrying about underlying representation language (4) SADL: a language to describe process models.
