Benefits and challenges of template-less
NL generation
Benefits:
• No additional step for programmer
• No opportunity to miss vital steps of the explanation
because there is no template
• Using the same interface for knowledge acquisition
provides a seamless interface for the expert
(more in KA CCE presentation)
Challenges:
• Knowledge engineer must write explicable code
(perhaps this is a benefit?)
• The system must be told the “correct” level of detail
