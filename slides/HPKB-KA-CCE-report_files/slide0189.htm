Experiments with the EXPECT PSM-Based
KA Scripts and NL Editor (Jim Blythe)
• These experiments test a KA tool that uses:
• PSM-based KA Scripts   [Blythe & Gil 99] that use
background knowledge about plan evaluation and
critiquing (a Problem-Solving Method + ontologies)
to guide the dialog with a user about a new critique
to be added
• An English-based editor  [Blythe & Ramachandran
99] that lets the user add or modify problem-solving
knowledge by manipulating its English paraphrase,
without having to see or understand the internal
formal representation
