• User adds a new method
• “In order to evaluate the force-ratio for a military task, check if the
required ratio for the task is less than the available ratio for the
task”.
• System points out that it is missing problem solving knowledge
• EMeD has analyzed interdependencies between the new method
and its sub-methods (for computing required ratio and for
computing available ratio) and found that the user has to add a
new method for computing available ratio.
• System proposes an initial sketch of missing knowledge
• EMeD predicts what the missing sub-method has to do and
proposes a capability : the new method should compute available
ratio for a given military task
• EMeD predicts the output of the missing sub-methods, and
proposes a result type : the new method should return a number
(force ratio) to be compared with another number (required force
ratio)
