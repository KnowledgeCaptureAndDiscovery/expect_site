DARPA High-Performance Knowledge Bases:
Challenge Problems
Challenge problems are posed by a 3rd party
Battlespace (AT), Crisis management (IET)
Year 1 BS problem: Enemy workarounds to target damage
Given: a possible target (e.g., bridge) whose damage could delay movement of an enemy unit
Output: assess delay to the unit’s movement for each possible workaround (e.g., ford river, install temp bridge, etc.)
Phase 1: test set within system scope.  Measured initial performance (baseline), and performance after repairing KB for a week (retest).
Phase 2: test set for problems that required new scope for the system. Also collect baseline and retest data.
Year 2 BS problem: COA critiquing
Finding flaws in a given COA
Not a bakeoff, rather all systems share knowledge and are integrated
