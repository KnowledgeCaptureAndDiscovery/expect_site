SHERPA’s Interdependency Models (IM)

  • When EXCALIBUR is used to develop the KBS a complete IM is created automatically

  • For KBS not developed with EXCALIBUR:

    • initial model can be created automatically
    • KBS developers can adapt it manually

  • Tools to support creation of IM:

    • IM creation via static analysis of declarative data structures in KBS
    • dynamic creation of IM, recording KBS runtime access to KB
    • translation-based IM creation when porting ontologies to target code
    • manual IM creation, can be partially supported using EXCALIBUR
